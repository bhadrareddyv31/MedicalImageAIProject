# Medical Image Diagnosis using AI 🧠🩺

A deep learning-based system to detect pneumonia and skin cancer from medical images using transfer learning (MobileNetV2). Includes model explainability via Grad-CAM.

## 🔧 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Grad-CAM (planned)
- Jupyter Notebooks
